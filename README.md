# StealthEra Website

A pixel-perfect recreation of the StealthEra website built with React, Vite, and Tailwind CSS.

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
